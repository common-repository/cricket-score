== Description ==
Cricket Live Score wordpress plugin allows you to add real-time cricket scores to your website for free. This plugin allows you to add live cricket data of all today's events by every well-known leagues, just live events that are happening right now or all yesterdays events.

Cricket live scores is added by a simple shortcode so you can add this events whenever you want - to a single page as a main content, to a single post, as a sitewide element in header or sidebar of you website as a widget etc.

When setting up plugin's display option please choose best option that fits your needs:
* Display cricket livescores as iframe containing ads
* Display cricket livescores as native code without ads but containing a link to developer's websites

Main Advantages of this plugin:
1. Plugin is fully responsive and can be added to any website
2. Plugin supports SSL/HTTPS so it's fully secured for your website
3. It's totally free for everyone
4. Information is updating in real time so users don't need to refresh the page to get the score
5. Automati—Å timezones changes for users from different countries so they see accurate date and time

And more to come!

= Interaction with External Services =
Through the external service fscore.in, the Cricket Live Score Widget provides users with real-time information on the schedule and results of cricket matches. Fscore.in is a reliable source of sports information, earning trust within the community.

The functionality of fscore.in is built on delivering real-time data for selected matches, including the match start time, current status, incidents, and score. This implies that data is automatically updated without the need for page reloads.

Such an approach ensures users quick and uninterrupted access to the latest sports data, eliminating the need for manual information updates and allowing them to stay informed about ongoing events in real-time.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Choose display option you prefer: with ads or without them
4. Get a shortcode you need (all today matches, live matches or yesterday matches)
5. Put this shortcode whenever you want: on a single page, post or in sitewide blocks like header, sidebar or footer. Use your imagination!
7. That's it! Now you have real-time cricket livescores on your website!
